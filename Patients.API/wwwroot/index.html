Usage Notes

Comma-separated relation fields accept values like 1,2,3.

Patient gender values map to 1 = Male, 2 = Female, 3 = Other.

If HTTPS certificates are not trusted in the browser, open the API once directly and accept the certificate before using this page.

Get the JWT from the Users API separately, for example via Swagger or your token endpoint, then paste it into the Access token box.

Patient write actions require an Admin role token because the controller uses [Authorize(Roles = "Admin")].
